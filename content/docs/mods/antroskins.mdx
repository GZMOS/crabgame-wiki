AntroSkins is a client-side mod by <Contributor id="antropomeda" /> that lets you load custom player models directly into Crab Game.

### Features

- Replace vanilla player models with custom rigged avatars (`.bundle` files)
- Tweak bone offsets (positions and rotations) in real-time and reset them to defaults
- Apply a single skin to everyone, or use **Chaos Mode** to distribute skins randomly across the lobby
- Exclude specific skins from the random pool
- Share configurations with others using a short share code
- Search and sort bones in the menu

The menu is toggled with **F6** by default and can be rebound directly in the menu.
